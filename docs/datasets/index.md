## API's

* <span style="color:rgba(21, 153, 87)">Diversas</span>
	- [Brasil API](https://brasilapi.com.br/){:target="_blank"}: Transformando o Brasil em uma API;
	- [Random Data API](https://random-data-api.com/documentation){:target="_blank"}: Gera dados pseudo-aleatórios para testar aplicações com API'S;
	- [Rapid API](https://rapidapi.com/){:target="_blank"}: Nesse site você pode encontrar mais de 10.000 API's públicas e 1 milhão de desenvolvedores ativos no local. As categorias disponíveis (free) encontram-se [aqui](https://rapidapi.com/collection/list-of-free-apis){:target="_blank"};
	- [Public APIs](https://github.com/public-apis/public-apis){:target="_blank"}: a plataforma agrupa API's em 40 categorias, tornando mais fácil navegar e encontrar a API certa para atender às suas necessidades;
	- [API list](https://apilist.fun/){:target="_blank"}: O site oferece mais de 500 API's Web, juntamente com informações detalhadas sobre como usá-las;
	- [Public REST APIs](https://documenter.getpostman.com/view/8854915/Szf7znEe){:target="_blank"}: Esta é uma coleção de diferentes API's REST que são completamente públicas e não requerem nenhuma autenticação, tornando mais fácil para os usuários brincar e entender o que são as API's, experimentando as muitas maneiras diferentes pelas quais as API's podem ser usadas;
	- [apipheny](https://apipheny.io/free-api/){:target="_blank"}: Lista gigantesca de API's publicas (no key);
	- [HG brasil](https://hgbrasil.com/){:target="_blank"}: Disponibiliza API's para dados climáticos, financeiros e localização por meio de IP's;
	- [Catálogo de APIs Governamentais](https://www.gov.br/conecta/catalogo/){:target="_blank"};
	- [Universities List](https://github.com/Hipo/university-domains-list-api){:target="_blank"}: API que fornece dados e sites de universidades no mundo;
	- [Google APIs Explorer](https://developers.google.com/apis-explorer/){:target="_blank"}: Catálogo de API's do Google;
	- [JSON placeholder](https://jsonplaceholder.typicode.com/){:target="_blank"}: Proporciona dados fictícios como fotos, publicações, comentários, entre outros para testar suas APP's;


* <span style="color:rgba(21, 153, 87)">Mapas, localizações e viagens</span>
	- [Google Maps](https://developers.google.com/maps/documentation?hl=pt-br){:target="_blank"}: A API do Google é Uma das mais usadas para mapas;
	- [Bing Maps](https://www.microsoft.com/en-us/maps/choose-your-bing-maps-api){:target="_blank"}: API da Microsoft;
	- [https://developer.apple.com/maps/](https://developer.apple.com/maps/){:target="_blank"}: API da Appple;
	- [Here Maps](https://developer.here.com/documentation){:target="_blank"}: Considerada uma das API's de mapas mais atualizadas do mundo;
	- [OpenStreetMaps](https://wiki.openstreetmap.org/wiki/API){:target="_blank"};
	- [Mapbox](https://docs.mapbox.com/){:target="_blank"}: Personaliza mapas digitais usando Here maps;
	- [OnWater](https://onwater.io/){:target="_blank"}: API para verificar se um determinado ponto geográfico está na água pou na terra;
	- [uebermaps API](https://uebermaps.com/api/v2){:target="_blank"}: Plataforma para que todos possam criar e compartilhar mapas com seus lugares favoritos;
	- [https://viacep.com.br/](https://viacep.com.br/){:target="_blank"}: Para consultar Códigos de Endereçamento Postal (CEP) de todo Brasil;
	- [Buscar CEP](https://buscarcep.com.br/site.php){:target="_blank"};
	- [WebmaniaBR](https://webmaniabr.com/docs/rest-api-cep-ibge/){:target="_blank"}: Para consulta de CEP e IBGE grátis diretamente nos Correios;
	- [Travelbriefing](https://travelbriefing.org/api){:target="_blank"}: Acesso a informações específicas do país sobre regulamentos de viagem, condições ambientais predominantes, saúde e vacinas, clima, requisitos de visto, moeda, tomadas elétricas e idioma;
	- [Walk Score](https://www.walkscore.com/professional/travel-time-http-api.php){:target="_blank"}: Calcula os tempos de viagem entre uma origem e um conjunto de destinos. Visualize os tempos de viagem em um mapa e suporte os tempos de caminhada, transporte público, direção e ciclismo;
	- [FlightStats Weather API](https://www.programmableweb.com/api/flightstats-weather){:target="_blank"}: Informações atualizadas sobre as condições climáticas atuais em um aeroporto;


* <span style="color:rgba(21, 153, 87)">Clima</span>
	- [OpenWeatherMap API](https://openweathermap.org/api){:target="_blank"};
	- [Climatempo](https://advisor.climatempo.com.br/){:target="_blank"}: A Climatempo disponibiliza através de uma API a previsão do tempo e outros dados meteorológicos em tempo real;
	- [CPTEC/INPE](http://servicos.cptec.inpe.br/XML/){:target="_blank"}: Dados da Previsão de Tempo, IUV e Ondas do CPTEC/INPE no formato XML puro;
	- [RainViewer](https://www.rainviewer.com/api.html){:target="_blank"}: Fornece acesso a dados de mapeamento de chuva com base em informações de radares meteorológicos globalmente;
	- [Wunderground](https://www.wunderground.com/login){:target="_blank"}: Previsão do tempo local e de longo alcance, boletins meteorológicos, mapas e condições climáticas tropicais para locais em todo o mundo em 80 idiomas;
	- [OpenUV](https://www.openuv.io/){:target="_blank"}: Busque a previsão global do índice UV em tempo real;	

	
* <span style="color:rgba(21, 153, 87)">Financeiras</span>
	- [CurrencyLayer](https://currencylayer.com/){:target="_blank"}: API com informação em tempo-real sobre 168 moedas ao redor do mundo;
	- [Open Exchange Rates](https://openexchangerates.org/){:target="_blank"}: Dados de taxa de câmbio ​​e conversão de moedas;
	- [APIS B3](http://www.b3.com.br/data/files/2B/41/CC/5D/10F42610D290A226790D8AA8/APIs-B3-Visao-Geral-versao-1.0.pdf){:target="_blank"}: A B3 disponibiliza APIs para permitir, de forma ágil, interação com o mercado;
	- [Nomics](https://nomics.com/docs/){:target="_blank"}: API com informações de criptomoedas e Bitcoin;
	- [Fixer](https://fixer.io/){:target="_blank"}: Para verificar a taxa de câmbio entre as diferentes moedas;
	- [Blockchain API](https://www.blockchain.com/api){:target="_blank"}: Receção de dados de mercado em tempo real, pedido de informações de saldo e realização de trocas.;
	- [CoinAPI](https://docs.coinapi.io/#md-docs){:target="_blank"}: Dados de mercado em 274 casas de câmbio integradas em uma única API.	
	- [CoinDesk](https://www.coindesk.com/coindesk-api){:target="_blank"}: Acesse dados do Índice de Preços do Bitcoin, que representa uma média dos preços do Bitcoin nas principais bolsas do mundo;
	- [Yahoo finance API](https://www.yahoofinanceapi.com/){:target="_blank"}: Alternativa da API do Yahoo Finance para mercado de ações, criptomoedas e câmbio;
