Introdução à biblioteca Pandas¶

  • Pandas uma das bibliotecas do Python mais completas para manipulao e anlise de dados;
  • O criador da biblioteca Wes McKinney. O nome da biblioteca faz referncia ao Panel Data (dados em painel, traduo livre), que refere-se a um tipo especial de dados estudados na Econometria;
  • Para instalar basta usar: pip3 install pandas;

Documentação¶

  • O site oficial da biblioteca https://pandas.pydata.org/ e a documentao oficial encontra-se disponvel aqui;
  • Uma folha de dicas para facilitar o uso do Pandas est disponvel em Folha de dicas;
  • No site oficial da biblioteca ainda disponibiliza uma lista de comandos para os usurios de linguagens como R, SQL, SAS ou STATA, possam comparar as sintaxes dos comandos com a mesma funcionalidade no Pandas;

Introdução à biblioteca Pandas¶

  • Os principais objetos da biblioteca Pandas so: Series e DataFrames;
  • Um objeto de tipo Series uma sequncia de valores que apresentam uma indexao. O objeto Series comparvel com o objeto lista do Python ou um arranjo unidimensional do NumPy, mas a indexao com um objeto Series diferente. No NumPy no h muito controle sobre os ndices de linha e coluna; mas com um objeto Series, cada elemento da srie deve ter um ndice, nome ou uma chave;
  • O ndice pode consistir em strings;
