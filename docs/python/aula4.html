Introdução à biblioteca Pandas¶

  • Pandas é uma das bibliotecas do Python mais completas para manipulação e análise de dados;
  • O criador da biblioteca é Wes McKinney. O nome da biblioteca faz referência ao Panel Data (dados em painel, tradução livre), que refere-se a um tipo especial de dados estudados na Econometria;
  • Para instalar basta usar: pip3 install pandas;

Documentação¶

  • O site oficial da biblioteca é https://pandas.pydata.org/ e a documentação oficial encontra-se disponível aqui;
  • Uma folha de dicas para facilitar o uso do Pandas está disponível em Folha de dicas;
  • No site oficial da biblioteca ainda disponibiliza uma lista de comandos para os usuários de linguagens como R, SQL, SAS ou STATA, possam comparar as sintaxes dos comandos com a mesma funcionalidade no Pandas;

Introdução à biblioteca Pandas¶

  • Os principais objetos da biblioteca Pandas são: Series e DataFrames;
  • Um objeto de tipo Series é uma sequência de valores que apresentam uma indexação. O objeto Series é comparável com o objeto lista do Python ou um arranjo unidimensional do NumPy, mas a indexação com um objeto Series é diferente. No NumPy não há muito controle sobre os índices de linha e coluna; mas com um objeto Series, cada elemento da série deve ter um índice, nome ou uma chave;
  • O índice pode consistir em strings;
